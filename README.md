# Interview Starter

This repository contains two API starter options for interview exercises:

- `nestjs/` - NestJS + TypeORM + SQLite
- `expressjs/` - Express + TypeScript + SQLite

Pick one stack and work inside that directory.

## Option 1: NestJS

Use this if you want a structured framework with modules, decorators, and built-in patterns.

- Folder: `nestjs/`
- Start command: `npm run start:dev`
- Docs: `nestjs/README.md`

Quick start:

```bash
cd nestjs
npm install
npm run start:dev
```

## Option 2: ExpressJS

Use this if you want a lightweight setup with minimal framework abstractions.

- Folder: `expressjs/`
- Start command: `npm start`
- Docs: `expressjs/README.md`

Quick start:

```bash
cd expressjs
npm install
npm start
```

## Notes

- Both projects run on `http://localhost:3000` by default.
- Each option includes its own tests and project-specific documentation.
